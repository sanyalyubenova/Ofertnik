# Импорт на данни от НСИ

## Стъпки за импортиране на данни за населените места в България

### 1. Изтегляне на данни от НСИ

1. Отворете сайта на НСИ: https://www.nsi.bg/nrnm/
2. Изтеглете данните в CSV формат
3. Запазете файла като `nsi-data.csv` (или друго име)

### 2. Проверка на формата на CSV файла

CSV файлът трябва да съдържа колони с:
- **Наименование** (или `Name`) - името на населеното място
- **Област** (или `Region`) - областта
- **Община** (или `Municipality`) - общината

### 3. Изпълнение на скрипта

```bash
node scripts/import-nsi-data.js path/to/nsi-data.csv
```

Или ако файлът е в корена на проекта:

```bash
npm run import-nsi nsi-data.csv
```

### 4. Резултат

Скриптът ще генерира файла `public/bulgarian-cities.js` с всички населени места от НСИ данните.

### 5. Тестване

Отворете приложението и тествайте автокомплит функционалността с новите данни.

## Забележки

- Скриптът автоматично премахва дублирани записи
- Ако имената на колоните в CSV файла са различни, редактирайте `convertToJavaScriptArray` функцията в `import-nsi-data.js`
- Може да се използва периодично за актуализация на данните

## Алтернатива

Ако CSV файлът е в различен формат, може да се използва библиотека като `csv-parser` или `xlsx` за по-добра обработка.

```bash
npm install csv-parser
```

След това модифицирайте скрипта да използва тази библиотека.
